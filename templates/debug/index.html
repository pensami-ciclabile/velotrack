Pagina nascosta con gli strumenti di diagnostica. Non è collegata da nessuna voce di menu: per raggiungerla bisogna conoscere l'URL. Hidden utility hub. Not linked from any menu — you have to know the URL to get here.

route

Ispettore tracceRide inspector

Visualizza ogni traccia GPX punto per punto. Cerchi colorati per velocità, tooltip con timestamp, lat/lon, distanza cumulata. Utile per trovare punti corrotti nei file sorgente. Browse every GPX ride point by point. Circles colored by speed, tooltips with timestamp, lat/lon and cumulative distance. Handy for spotting corrupted points in source files.

layers

Sovrapposizione lineeLine overlay

Seleziona una linea e vedi tutte le tracce che la compongono sulla stessa mappa, ciascuna in un colore diverso. Utile per capire quale singola traccia sta sballando la media aggregata. Pick a line to see every contributing ride on the same map, each in a distinct color. Handy for spotting the one ride that's dragging the aggregate average off.

traffic

Mappa semaforiTraffic lights map

Tutti i semafori registrati su una mappa interattiva. Consultazione sola lettura; per aggiungere o rimuovere semafori usa uv run main.py traffic-lights --watch. All recorded traffic lights on an interactive map. Read-only view; to add or remove lights run uv run main.py traffic-lights --watch.
